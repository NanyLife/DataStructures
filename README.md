# DataStructures


**Lab 1**

*Время < 1 с*  

*Память < 64 Mb*

*Task:* Дано натуральное число N. Требуется представить его в виде суммы двух натуральных чисел А и В таких, что НОД (наибольший общий делитель) чисел А и В - максимален.
Входные данные
Во входном файле записано натуральное число N (2<N<103)
Выходные данные
В выходной файл выведите два искомых числа А и В. Если решений несколько, выведите любое из них.

*Примеры*

| ***входные данные*** |  ***выходные данные*** | 
|---------|---------|
|15  | 5 10 |

| ***входные данные*** |  ***выходные данные*** | 
|---------|---------|
| 16 | 8 8 |
